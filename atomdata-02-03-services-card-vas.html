VAS (Value-Added Services)

оставить заявку
Описание

Value-Added Services (VAS) — дополнительный спектр Услуг, которые предлагаются к основным для более полного комплексного решения задачи Клиента. Такого рода сервисы могут быть организованы как силами Клиента, там и предоставлены сервис-провайдером, что часто достаточно удобно и экономически выгодно. Эти типы сервисов не являются необходимыми для основной информационной системе, размещаемой в облаке, но могу привнести продукту добавочную ценность, удобство и гибкость пользования, а также перенести ответственность за их функционирование в зону ответственности облачного оператора.

Услуги VAS также могут приносить доход и самому Заказчику, т.е. быть монетизированы: их можно предлагать конечным клиентам в виде коммерческих дополнительных сервисов, прибыль от которых будет полностью или частично покрывать расходы на них. Такого рода Услуги, как правило, не являются основной затратной статьёй в общем бюджете облачного проекта, просты в подключении и не должны влиять на основной функционал информационной системы при отключении VAS.

Преимущества
К преимуществам сервисов VAS для основных сервисов Заказчика, размещаемых в облачной инфраструктуре АТОМДАТА, можно отнести:
Простая процедура подключения и отключения
Понятная и прозрачная единица тарификации
Возможность получить Услугу на тестовый период (к примеру, для обоснования возможности монетизации Услуги)
Отсутствие у Заказчика необходимости наличия компетенций в области Услуг VAS (Заказчику достаточно принять решение о необходимости Услуги без углубления в механизмы организации её архитектуры)
Примеры использования
Типовые сценарии использования сервисов VAS в облаке ATOMDATA Cloud:
1.
Приобретение права использования лицензионного программного обеспечения по подписочной модели с возможностью динамического увеличения и уменьшения количества лицензий (без капитальных затрат)
2.
Аренда сетевых устройств (виртуальных и / или физических) для построения требуемой сетевой связности (в том числе сертифицированных ФСТЭК)
3.
Размещение дополнительного специализированного оборудования в ЦОД АТОМДАТА
4.
Мультифакторная аутентификация авторизованных администраторов или конечных пользователей Заказчика при процедуре подключения к порталу управления облаком для повышения уровня информационной безопасности
Клиентские кейсы:
Задача
Организация удалённых рабочих столов сотрудников банка на базе решения Virtual Desktop Infrastructure (VDI) с безопасным доступом с домашних рабочих компьютеров.
Состав решения
Частное облако ATOMDATA Cloud с развёрнутой инфраструктурой VDI. Сервис многофакторной аутентификации с вторым фактором приложения вендора.
Результат
После организации доступа сотрудников к основному сервису VDI c доступом по паре логин/пароль, был подключен сервис многофакторной аутентификации, который в первый месяц работы запретил подключение к удалённым рабочим столам около 200 попыткам подключения без подтверждения второго фактора аутентификации.
Итог проекта
Помимо основного функционала VDI, была повышена безопасность подключения.
Задача
Сезонное масштабирование мощностей продающего сайта.
Состав решения
Публичное облако ATOMDATA Cloud для временного размещения (2‑4 месяца) дополнительных серверов приложений и серверов СУБД с временной арендой лицензий для отработки высокого трафика конечных клиентов.
Результат
Организована временная зона — пул серверов приложений и серверов СУБД с лицензионным системным и платформенным ПО с возможностью их выключения без оплаты аренды лицензий в низкий сезон.
Итог проекта
Наличие готовой к запуску в любой момент инфраструктуры для отработки высокого сезона спроса на товары Клиента без капитальных затрат на покупку лицензионного ПО.
Вопросы / ответы
Как тарифицируются основные VAS-сервисы?

Услуги облачных VAS-сервисов ATOMDATA Cloud оплачиваются по фиксированному тарифу (flat rate) в месяц. Основные единицы тарификации:

  • Пользователь, штуки
  • Виртуальная машина, штуки
  • Виртуальное ядро vCPU, штуки
  • Сетевое устройство, штуки
Возможно ли получить в тест услуги VAS и каков допустимый срок тестирования?
Да, это возможно без оплаты на срок тестирования. Общепринятый срок тестирования составляет 2 календарных недели, но он может быть незначительно увеличен по договорённости с менеджером по продажам.
